# Text scaling support in Chrome Canary

## AI Summary
This post discusses a new feature in Chrome Canary that enables web pages to respect system-wide text scaling settings on iOS and Android. Traditionally, changing accessibility text size settings on mobile devices hasn't affected web content, but the new `<meta name=text-scale>` tag addresses this gap. Unlike page zoom, this feature allows developers to scale text specifically without altering layout or other elements, improving accessibility for users with visual needs.

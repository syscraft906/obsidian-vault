# Angular support for generating apps in Google AI Studio is now available

## AI Summary
The Angular team has integrated support for generating Angular applications using Google AI Studio, leveraging the Gemini model to accelerate prototyping and development workflows. Developers can now create Angular (TypeScript) apps directly through AI Studio's interface, export code to GitHub, deploy via Cloud Run, and share projects. This integration simplifies rapid app generation, AI-powered feature development, and streamlined deployment, enhancing Angular's modern tooling ecosystem.

# Your menu doesn’t need Miller’s 7±2 rule

## AI Summary
This post clarifies that Miller’s 7±2 rule was originally about short-term memory capacity, not the optimal number of menu items in UI design. It critiques the common misapplication of the rule to digital menus and advises UX designers to focus on the context of the original cognitive psychology research. The piece emphasizes understanding research foundations to avoid design myths.

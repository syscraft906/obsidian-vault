# Modal vs. Separate Page: UX Decision Tree

## AI Summary
This article explores the UX trade-offs between modals and new page navigation, providing guidelines for choosing the right interface pattern. It distinguishes between modals, overlays, and nonmodals, emphasizing context preservation for quick tasks and full-page navigation for complex workflows. The post outlines a 4-step decision tree framework to evaluate user context, task complexity, background data reference needs, and overlay types. Key takeaways include avoiding modals for repetitive tasks and using nonmodals by default except when preventing critical user errors.

# Smashing Animations Part 8: Theming Animations Using CSS Relative Colour

## AI Summary
This blog post explores using CSS relative colour values and the OKLCH colour space to theme and animate graphics dynamically. It details how a single 'foundation' colour can generate entire palettes through chroma, lightness, and hue adjustments, simplifying design systems and enabling smooth animations. The author demonstrates techniques for creating reusable color relationships in SVGs, gradients, and interactive elements, with examples of animated lighting effects and theming for different times of day or seasonal themes.
